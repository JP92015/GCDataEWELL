Source Data: UCI HAR Dataset by Human Activity Recognition Using Smartphones Dataset. Version 1.0 by Jorge L. Reyes-Ortiz, Davide Anguita, Alessandro Ghio, Luca Oneto.
The data contain two separate datasets organized in separate folders for training and testing as text data. In each Folder there is a Feature Vector table X_*.txt, an activity row_ID y_*.txt, and subject row ID subject_*.txt table. Inertial signals are stored in sub_folders and were not used in this analysis.
There are three general information files:
'features_info.txt': shows information about the variables used on the feature vector.
'features.txt': lists all features.
'activity_labels.txt': links the class labels with activity names.
For more detail please consult: http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones#.;
